## ✨ Overview

**AI Translator Pro** is a production-grade language translation application built using Python and modern web technologies.

It combines:

- ⚡ Real-time translation (AJAX, no page reload)
- 🎙 Speech-to-text (microphone input)
- 🔊 Text-to-speech (browser & Google TTS)
- 🌗 Dark / Light theme support
- 🎛 Adjustable speech speed
- ♻ Smart audio caching
- 💎 Professional two-panel SaaS UI

This project demonstrates full-stack architecture, API integration, modern frontend engineering, and speech processing systems.

# 🖥 Desktop Application

Built using **Tkinter + Google TTS (gTTS)**

### Features
- Clean GUI Interface
- Microphone input (SpeechRecognition)
- High-quality multilingual voice output
- Auto-Speak mode
- Smart audio caching
- Loading status indicator
- Dark mode toggle

# 🌐 Web Application

Built using **Flask + JavaScript Web Speech API**

### Features
- Real-time translation using AJAX
- No page reload architecture
- Speech input via browser microphone
- Native browser text-to-speech
- Voice speed control
- Auto-Speak functionality
- Language swap feature
- Copy-to-clipboard support
- Professional SaaS layout
- Dark / Light theme toggle
- Responsive UI design

# 🧠 Architecture
Frontend (HTML/CSS/JS)
↓
AJAX Request
↓
Flask Backend (Python)
↓
Google Translation API
↓
JSON Response
↓
Dynamic UI Update

Speech systems:
- Browser Web Speech API (Web)
- Google Text-to-Speech (Desktop)

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Priyatham27/AI-Language-Translator-Pro.git
cd AI-Language-Translator-Pro
```

## 2️⃣ Run Web Application

```bash
cd web_app
pip install -r requirements.txt
python app.py
```

## 3️⃣ Run Desktop Application

```bash
cd desktop_app
pip install -r requirements.txt
python main.py
```
